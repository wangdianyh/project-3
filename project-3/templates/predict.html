Pokémon Battle Predict

Choose 2 Pokémon from the dropdown list to check the battle result prediction

Pokémon battle prediction is a data model based on the theory of marchine learning and mathsmatics probability events calculation. After analyzing [50,000] Pokémon battle instances of [783] Pokémon in the database, I have developed this amazing data model to predict battle result between 2 Pokémon, which has reached nearly [95%] accurate prediction in average now.

Furthermore, this data model can improve the accuracy of prediction by gaining [battle instances], which means with the increasing of battle instances gaining in the database, the accuracy of battle prediction has the possibility to improve from the current [95%] to as close as [100%] infinitely.

For more details of data model development and error testing precess, please visit author's raw ipynb file on Github.
